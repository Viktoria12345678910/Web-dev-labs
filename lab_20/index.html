Валідація номера банківської карти

16 цифр, починається з 4 або 5.

function validateCard(num) {
  const regex = /^[45]\d{15}$/;
  return regex.test(num.replace(/\s/g, ''));
}

Валідація електронної пошти

Лише латинські літери, цифри, "_" і "-". Символ "-" не може повторюватися. Не може починатися з "_" або "-".

function checkEmail(email) {
  // починається з літери/цифри, тіло: літери/цифри/_/-, без --
  const regex = /^[A-Za-z0-9][A-Za-z0-9_]*(-[A-Za-z0-9_]+)*@[A-Za-z0-9-]+\.[A-Za-z]{2,}$/;
  return regex.test(email);
}

Валідація URL

Рядок починається з http, https або ftp, потім двокрапка і два слеші.

function isValidURL(url) {
  const regex = /^(https?|ftp):\/\/.+/;
  return regex.test(url);
}
